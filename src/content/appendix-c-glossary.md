# Appendix C: Glossary

| Term | Definition |
|------|-----------|
| Stage | A single maze level that the player navigates from entry to exit |
| Run | A complete session of consecutive stages from start to death or completion |
| Template | A pre-built stage layout stored in ServerStorage and spawned at runtime |
| Threat Tier | Enemy difficulty classification (Tier 1 through Tier 4) |
| Damage Affinity | The interaction between attack types and enemy resistances/weaknesses |
| Fixed Camera | Camera mode locked to a predefined position for boss/puzzle rooms |
| Smooth Follow | Default camera mode that tracks the player through corridors |
| Core Loop | The repeating gameplay cycle within a single stage (enter → fight → loot → exit) |
| Meta Loop | The progression systems that drive engagement across multiple sessions |
| Battle Pass | Seasonal progression track with free and premium reward tiers |
| Ghost Run | A replay of another player's run visible as a transparent overlay |
| DevEx | Developer Exchange: Roblox's system for converting Robux to real currency |
| MVP | Minimum Viable Product: the first publishable version of the game |
| Sprint | A 2-week development cycle with defined goals and deliverables |
| Hub / Lobby | The social space where players gather between dungeon runs |
| Telegraphing | Visual and audio cues that warn players of incoming attacks or hazards |
| Server-Authoritative | Game logic that runs on the server to prevent cheating and ensure consistency |
| DataStore | Roblox's built-in persistent storage system for saving player data |
| RemoteEvent | Roblox's system for sending data between server and client scripts |
| Invulnerability Window | Brief period after taking damage where the player cannot be hit again |
| AoE | Area of Effect: an attack that damages all enemies within a radius |
| D1 / D7 / D30 | Retention metrics measuring what percentage of players return after 1, 7, or 30 days |
| ARPDAU | Average Revenue Per Daily Active User |
| DAU | Daily Active Users |
| Robux | Roblox's virtual currency used for in-game purchases |
